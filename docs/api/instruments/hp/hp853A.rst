Depending on the plugin the following frequency range is covered:

=======  ===============
Plugin   Frequency range
=======  ===============
HP8557A  10 kHz - 350 MHz
HP8558A  100 kHz - 1500 MHz
HP8559A  10 MHz - 21 GHz
=======  ===============
